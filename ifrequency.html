iFrequency

App to conversion frequency.

About project

To pass subject in my school, 50% attendance is required. Electronic class register used by my school can't conversion frequency on percents and can't presents frequency in tables. Manual counting was problematic for me and my class mates, soo I decided to change this and I create my own representation of data from class register.

How it works?

In first step, my app goes to login website of class register of my school, make screenshot and crop captcha image. When user fill all data fields correctly, the application logs in to service and goes to the student attendance.



Based on selected by user time range, the application parses data. Then, recalculates collected data into percent of attendance. Finally, creates a table with all stuff.
